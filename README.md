# Louche

---

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'louche'
```

## Usage

Louche provides a few validators to use in your ActiveModel/ActiveRecord classes:

* `EmailValidator`
* `URLValidator`
* `PhoneNumberValidator`
* `PostalCodeValidator`
